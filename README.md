Project Description:
LAW Snake & Ladders is an educational web-based game that spreads legal awareness using a
fun Snake & Ladder board concept. Snakes represent legal violations while ladders represent good
legal practices.

Tech Stack
HTML5, CSS3, JavaScript, VS Code, Web Browser

Features
• Interactive login and category selection
• Law learning through gameplay
• Snake & Ladder based logic
• Offline usable frontend project

Architecture
User → Login → Category → Law Pages → Game Board
